# 🗑️ Garbage Identification Project

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/choudharysidra240-ctrl/Garbage-identification/blob/main/notebooks/garbage_detection.ipynb)

## Overview
This project uses YOLOv8 for real-time garbage detection and classification.

## Quick Start
```bash
git clone https://github.com/choudharysidra240-ctrl/Garbage-identification.git
cd Garbage-identification
pip install -r requirements.txt
python app.py
```

## Files
- `garbage_detector.py` - Main Python script
- `app.py` - Gradio web interface
- `notebooks/garbage_detection.ipynb` - Jupyter notebook
- `taco_consolidated/data.yaml` - Dataset configuration
